get openvpn conf and keys

client.ovpn client.conf client.key client.crt ca.crt stunnel.conf

how to use:

1. install stunnel

i. download stunnel from official site: https://www.stunnel.org/downloads.html for windows: https://www.stunnel.org/downloads/stunnel-5.09-installer.exe

ii. install stunnel

iii. download stunnel.conf

ix. replace same name file in stunnel directory(C:\Program Files\stunnel\)

x. restart stunnel

1. install openvpn

i. download openvpn from official site: https://openvpn.net/index.php/open-source/downloads.html for 32-bit windows client: http://swupdate.openvpn.org/community/releases/openvpn-install-2.3.6-I001-i686.exe

ii. install openvpn

iii. download client.ovpn, client.key, client.crt,ca.crt

ix. put client.ovpn openvpn configure directory(C:\Program Files\OpenVPN\config\)

x. make a keys/ directory under C:\Program Files\OpenVPN\config\, and put client.keys client.crt ca.crt into it

xi. start openvpn. use client.conf to connect
